# Pokemon CLI

## Using uv

You can run with uv and it will install requirements for you:

```bash
uv run main.py [argument]
```

<br>

## Using pip

With pip, make sure to install requirements using the requirements.txt provided first:

```bash
pip install -r requirements.txt
```

Then you execute the program like any other python script:

```bash
python3 main.py [argument]
```

<br>

💡 _Replace [argument] with the name of a Pokémon (e.g., pikachu, charizard)._
